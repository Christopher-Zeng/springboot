Changes that were required to upgrade projects to Springboot 2.1.9.

Not all projects contain all of the below code or all of the below files, 
so not all projects require all changes.

After making changes, right-click project and run "Maven"->"Update Project"

You may need to set the time zone in MySQL, like this, for example:

SET GLOBAL time_zone = '+1:00';  

----------------

Change all occurrences of "new PageRequest" to "PageRequest.of"

----------------


App.java: organise imports to fix incorrect imports

	Replace: 
	@Bean
		EmbeddedServletContainerCustomizer errorHandler() {
			return new EmbeddedServletContainerCustomizer() {
	
				@Override
				public void customize(ConfigurableEmbeddedServletContainer container) {
					container.addErrorPages(new ErrorPage(HttpStatus.FORBIDDEN, "/403"));
				}
	
			};
		}
		
	With: 
	
	@Configuration
	     public class ServerConfig {
	             @Bean
	             public ConfigurableServletWebServerFactory webServerFactory() {
	                     TomcatServletWebServerFactory factory = new TomcatServletWebServerFactory();
	                     
	                     factory.addErrorPages(new ErrorPage(HttpStatus.FORBIDDEN, "/403"));
	                     return factory;
	             }
	     }
	     
ErrorHandlerController.java:
	organise the imports.


StatusUpdateService.java: 

	change "delete" to "deleteById"

	change: return statusUpdateDao.findOne(id);
	to: return statusUpdateDao.findById(id).get();
	
SearchService.java:
	change: return results.map(converter);
	to: return results.map(p -> new SearchResult(p));
	
	You can delete the now-unused "converter".
	
UserService.java:
	change:  userDao.findOne(id);
	to: userDao.findById(id).get();

StatusTest.java:

	change: 
	@SpringApplicationConfiguration(App.class)
	@WebAppConfiguration
	
	to:
	@SpringBootTest(webEnvironment=WebEnvironment.RANDOM_PORT)
	@TestPropertySource(locations="classpath:test.properties")
	
	change: statusUpdateDao.findOne(status.getId());
	to: statusUpdateDao.findById(status.getId()).get();
	
create the file "test.properties" in the resources directory.
